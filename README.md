# 🎠 WonderLand Nursery Registration Form

<form to register children in WonderLand Nursery with bilingual support (Arabic/English), PDF generation, and WhatsApp integration.>

## ✨ المميزات | Features

- 🌐 **ثنائي اللغة**: تبديل فوري بين العربية والإنجليزية مع دعم RTL
- 📊 **شريط تقدم تفاعلي**: يوضح للمستخدم مرحلة التسجيل
- 🎂 **حساب العمر تلقائياً**: من تاريخ الميلاد
- ✍️ **توقيع رقمي**: على الشاشة باللمس أو الماوس
- 📄 **توليد PDF احترافي**: يتم تحميله تلقائياً
- 📱 **إرسال عبر واتساب**: رسالة جاهزة مع رابط لإرفاق الـ PDF
- 📍 **روابط مباشرة**: فيسبوك + موقع الحضانة على خرائط جوجل
- 📱 **متجاوب**: يعمل على الموبايل، التابلت، والكمبيوتر

## 🚀 النشر على GitHub Pages

### الطريقة 1: يدوياً (الأسهل)
1. أنشئ مستودع جديد على GitHub باسم `wonderland-nursery-form`
2. ارفع ملف `index.html` إلى المستودع
3. اذهب إلى **Settings** → **Pages**
4. تحت **Build and deployment**، اختر المصدر: `Deploy from a branch`
5. اختر الفرع: `main` والمجلد: `/ (root)`
6. اضغط **Save**
7. بعد دقيقة، سيكون رابط موقعك:  
   `https://YOUR_USERNAME.github.io/wonderland-nursery-form/`

### الطريقة 2: باستخدام GitHub Actions (تلقائي)
1. أنشئ مجلد `.github/workflows/` في مستودعك
2. أضف ملف `deploy.yml` بالمحتوى الموجود في هذا المستودع
3. عند كل `git push`, سيتم النشر تلقائياً

## ⚙️ التخصيص

### تغيير رقم الواتساب
افتح `index.html` وابحث عن:
```javascript
const nurseryPhone = "201000000000"; // ضع رقمك هنا
```
غيّر الرقم إلى رقم الحضانة بصيغة دولية **بدون +** (مثال: `201012345678`)

### تغيير الروابط
- **فيسبوك**: ابحث عن `https://www.facebook.com/share/1DVNmrvTUs/` واستبدله برابط صفحتك
- **خرائط جوجل**: ابحث عن `https://maps.app.goo.gl/zmoaQUtjm32g4kSi8` واستبدله برابط موقعك

### تغيير الألوان
في قسم `<style>` في الأعلى، عدّل متغيرات `:root`:
```css
:root {
  --primary: #FF6B6B;    /* اللون الأساسي */
  --secondary: #4ECDC4;  /* اللون الثانوي */
  /* ... إلخ */
}
```
